  • MIDITimeTableCellID

    Stable identity for a time table cell. Unlike MIDITimeTableCellIndex — which is only a snapshot of a cell’s current (row, array-position) and goes stale the moment any cell in the same row is added, removed, or reordered — a cell’s id never changes across edits, so it’s the reliable way to track “the same cell” through a sequence of moves, resizes, or splits.
