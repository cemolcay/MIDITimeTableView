MIDITimeTableCellRepresentable

public protocol MIDITimeTableCellRepresentable

Optional protocol for app-owned cell models that can be displayed in a MIDITimeTableView.

MIDITimeTableView does not require this protocol; its data source remains the source of truth. Conforming your own model can make the data source implementation and app-side history code more structured without forcing your data to inherit from framework-owned model classes.
