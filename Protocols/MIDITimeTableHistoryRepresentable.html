MIDITimeTableHistoryRepresentable

public protocol MIDITimeTableHistoryRepresentable

Optional protocol for app-owned history stacks used with MIDITimeTableViewDelegate.midiTimeTableViewShouldPushHistory(_:).

The core view intentionally does not own history snapshots because app models may include arbitrary domain data. This protocol provides a small, consistent shape for apps that want to keep their own undo/redo manager near their time table data.
