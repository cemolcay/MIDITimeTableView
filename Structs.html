  • MIDITimeTableViewEditedCellData

    Edited cell data. Holds the edited cell’s stable id and its index before editing, and new row index, position and duration data after editing.

    Prefer id to locate the cell in your own data going forward — it stays valid regardless of how many other cells were added, removed or reordered since. index is a snapshot of the cell’s (row, array-position) at the moment the edit was reported and can go stale the instant another edit in the same batch shifts it.
